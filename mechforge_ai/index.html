极致解耦，3 秒上手

独家模块化架构，想查手册？装 50MB 轻量包即可；需要算力？再装 work 包。

# 安装 MechForge
$ pip install mechforge-ai
# 安装可选模块
$ pip install mechforge-ai[work] # CAE 工作台
$ pip install mechforge-ai[rag] # RAG 向量搜索
# 启动方式
$ mechforge # AI 对话模式
$ mechforge -k # 知识库模式
$ mechforge-work # CAE 工作台
